# Statistics & Probability

## Core Concepts
- **Descriptive Statistics:** Mean, Median, Mode, Variance, Standard Deviation, Skewness, Kurtosis.
- **Probability:**
  - Bayes' Theorem.
  - Conditional Probability.
  - Independence vs Mutually Exclusive.
- **Distributions:** Normal (Gaussian), Binomial, Poisson, Bernoulli, Uniform. Central Limit Theorem.
- **Inferential Statistics:**
  - Hypothesis Testing (Null vs Alternative hypothesis).
  - Type I Error (False Positive) and Type II Error (False Negative).
  - p-values, Confidence Intervals, Statistical significance.
  - Statistical Tests: t-test, ANOVA, Chi-Square test.

## Common Questions
1. Explain the Central Limit Theorem and why it is important.
2. What is a p-value? Explain it to a non-technical person.
3. If a disease affects 1% of the population, and a test is 99% accurate, what is the probability you have the disease if you test positive? (Bayes Theorem application)
