# Deep Learning Interview Questions & Topics

## Core Concepts

- **Fundamentals:** Perceptron, Multi-Layer Perceptron (MLP), Activation Functions (Sigmoid, Tanh, ReLU, Leaky ReLU, Softmax).
- **Optimization:** Backpropagation, Chain Rule, Optimizers (SGD, Momentum, RMSprop, Adam), Learning Rate schedules.
- **Loss Functions:** Mean Squared Error (MSE), Binary Cross-Entropy, Categorical Cross-Entropy.
- **Regularization (Preventing Overfitting):** Dropout, L1/L2 Regularization, Batch Normalization, Early Stopping.
- **Convolutional Neural Networks (CNNs):** Convolution layers, Pooling (Max, Average), Stride, Padding, Receptive Field, common architectures (ResNet, VGG).
- **Recurrent Neural Networks (RNNs):** Vanishing/Exploding gradient problems, LSTMs, GRUs.
- **Attention & Transformers:** Self-attention mechanism, Multi-head attention, Positional encoding, Encoder-Decoder architectures.

## Common Questions

1. Why do we use non-linear activation functions?
2. What causes the vanishing gradient problem and how do LSTMs solve it?
3. Explain the difference between Batch Normalization and Layer Normalization.
