# Supervised Learning Interview Questions & Topics

## Core Concepts
- **Linear Regression:** Assumptions, cost function, gradient descent, ordinary least squares (OLS), R-squared vs Adjusted R-squared.
- **Logistic Regression:** Sigmoid function, cross-entropy loss, odds ratio.
- **Support Vector Machines (SVM):** Margin, hyperplanes, kernel trick (linear, poly, RBF), soft margin vs hard margin.
- **Decision Trees:** Entropy, Information Gain, Gini Impurity, pruning, handling categorical variables.
- **Ensemble Methods:** 
  - Bagging (Random Forest): out-of-bag error, feature importance.
  - Boosting (AdaBoost, Gradient Boosting, XGBoost, LightGBM): sequential training, learning rate.
- **K-Nearest Neighbors (KNN):** Distance metrics (Euclidean, Manhattan), choosing K, curse of dimensionality.
- **Naive Bayes:** Bayes' theorem, conditional independence assumption, variations (Gaussian, Multinomial).

## Common Questions
1. What is the difference between generative and discriminative models?
2. Explain the Bias-Variance tradeoff.
3. How do you handle imbalanced datasets in classification?
