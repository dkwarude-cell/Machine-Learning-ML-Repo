# Machine Learning and Implementation Repo

This repository covers the complete structured implementation plan for understanding and implementing core Machine Learning concepts. Look into each directory to find stub files which contain explanations and logic notes in python comments. You can begin implementing the code directly under them!

## Structure:
### Overview:
- `00_Introduction/libraries_and_applications.py`: A quick intro to important ML libraries, datasets, and applications.
  
### Data:
- `01_Data_Preprocessing/data_preprocessing.py`: How to handle missing data, outliers, scaling, and train-test splits.

### Supervised Learning:
- `02_Supervised_Learning/Regression/regression_algorithms.py`: Linear, Multiple, Polynomial, Ridge, Lasso.
- `02_Supervised_Learning/Classification/classification_algorithms.py`: Logistic, KNN, SVM, Decision Tree, Random Forest, Naive Bayes, etc.

### Unsupervised Learning:
- `03_Unsupervised_Learning/unsupervised_algorithms.py`: K-Means, PCA, Hierarchical clustering.

### Evaluation and Metrics:
- `04_Evaluation_Metrics/evaluation_metrics.py`: Accuracy, Confusion Matrix, AUC/ROC, MSE/MAE, etc.

### Core Trade-offs & Model Improvements:
- `05_Regularization/regularization.py`: Dealing with Overfitting - L1/L2.
- `06_Cross_Validation/cross_validation.py`: K-Fold, Holdout mechanisms.
- `07_Model_Tuning_and_Concepts/model_concepts.py`: Bias vs Variance, Underfitting vs Overfitting.
- `08_Optimization_Algorithms/optimization_algorithms.py`: Gradient Descent, SGD, Adam.
- `09_Ensemble_Learning/ensemble_learning.py`: Bagging, Boosting, Random Forest, AdaBoost.
